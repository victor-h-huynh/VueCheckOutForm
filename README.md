# "Checkout Form" - Vue.js Exercise
=====================

# Overview

Reconstructing a UX-design from scratch while learning Vue.js

## Project setup
```
1. Fork this repository, then clone your fork of this repository.
2. npm install
3. npm run serve
4. open http://localhost:8080
```
